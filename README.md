Link do projeto - https://github.com/VictoorLDS1/NTime

Projeto para uma lista de tarefas proposta na matéria de Fábrica de software.

A ideia foi desenvolver uma listagem simples, no qual qualquer usuário possa facilmente gerir como se fosse um bloco de notas. O seu diferencial está no visual chamativo, com features voltadas para coloração e identificação de cada tarefa visualmente.
